# Overview

Stack is a self-funded AI trading system designed to operate autonomously and grow its own capital over time.

## Mission

Build a system that:
- identifies opportunities
- executes trades
- manages risk
- compounds profits

## Core Idea

Instead of relying on outside funding, Stack sustains itself.

Profits generated are reinvested back into the system, allowing continuous growth.

## Current Focus

- architecture design
- risk controls
- transparency
- public build process
